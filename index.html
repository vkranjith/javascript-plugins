JQuery - Form Validation

Introduction

This is a JQuery based form validation plugin. It's still under development phase and I know there are many form validation plugins out there. But with this plugin I wish to provide the maximum flexibility. This plugin outputs an object with all the validation error messages which can be used with any other third-party plugins (like jquery modal) to display the error messages.
